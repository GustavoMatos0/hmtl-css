Negrito / Destaque

Itálico / Ênfase

Nesta frase, temos um termo em itálico usango a tag <i></i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em></em>(semântica)

Texto Marcado

Podemos criar um texto marcado usando a tag <mark></mark>

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste paragrafo usando as tags <big></big>(obsoleto) e <small></small>

Texto Deletado

Podemos marcar um texto como EXCLUIDO para indicar que deve ser lido, mas não considerado. Usando a tag <del></del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ÊNFASE e indicar que ele foi adicinado depois usando a tag <ins></ins>

Existe tambem u sublinhado com a tag <u></u>(não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup></sup>

Texto Subscrito
